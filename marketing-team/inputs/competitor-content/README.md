# Competitor Content — Input folder

Drop one folder per competitor with the source material the Competitor Researcher agent will analyze.

## Convention

```
competitor-content/
  igor-dias/
    2026-05-12_post-comp-bigtech.png       # screenshot of post
    2026-05-12_post-comp-bigtech.url       # plain text file with LinkedIn post URL
    notes.md                               # optional: your observations
  lipe-boff/
    ...
  lenny-rachitsky/
    ...
```

## What to include weekly

- 3-7 posts per competitor that performed well (visible high engagement)
- 1-2 posts per competitor that flopped (if visible) — useful to learn what NOT to do
- Any new account/competitor you're tracking

## What to skip

- Old screenshots (>30 days) — clutter
- Posts already analyzed in a previous cycle (mark them in notes.md to avoid re-analysis)
- Generic reposts of other people's content (low signal)

## Hard rule

Every post analyzed must have a verifiable source (screenshot OR live URL). The Competitor Researcher agent will reject inputs without source attribution.
